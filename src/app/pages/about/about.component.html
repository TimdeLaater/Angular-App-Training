Over deze applicatie

Deze applicatie is gemaakt in het kader van het vak 'Clientside web frameworks' van Avans Hogeschool Breda. Github: https://github.com/TimdeLaater/Angular-App-Training

Naam van jouw project

Mijn applicatie gaat als volgt werken er staan oefeningen al in een database en deze kun je selecteren op spiergroep. vervolgens kan de gebruiker een Trainingschema aanmaken waarin vervolgens trainingen toegevoegd kunnen worden. Deze trainingen bestaan weer uit een collectie van oefeningen
